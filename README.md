## РГР — АВМО (вариант 1): Жордан–Гаусс + симплекс

Программа решает задачу линейного программирования в **канонической форме**:

- ограничения
- целевая

Вход задаётся матрицей размера (m+1)*(n+1):

- первые `m` строк: коэффициенты A и последний столбец — b
- последняя строка: коэффициенты c и последний элемент (обычно 0) — свободный член целевой функции

Все вычисления ведутся в **рациональных числах** (`fractions.Fraction`), то есть без ошибок округления.

### Запуск

Из папки:

```bash
python main.py path/to/input.txt
```

### Минимизация / максимизация

В первой значимой строке файла можно указать режим:

- `max` — максимизация (по умолчанию, если строка отсутствует)
- `min` — минимизация

Далее идёт матрица (m+1)*(n+1) как обычно.

### Выход

Программа печатает:

- статус (оптимум / нет допустимых решений / неограничена сверху)
- оптимальный план x и значение Z
- если оптимальных решений бесконечно много — **общий вид** (как выпуклая комбинация нескольких оптимальных опорных решений)
